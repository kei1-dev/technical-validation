ベクトルバケットの一覧表示

注記

Amazon S3 Vectors は Amazon Simple Storage Service のプレビューリリースであり、変更される可能性があります。

Amazon S3 コンソールまたは AWS CLI を使用して、すべてのベクトルバケットを表示できます。一覧表示の操作はプレフィックスベースのフィルタリングをサポートしているため、アカウントに多数のベクトルバケットがある場合に特定のバケットを見つけるのに役立ちます。ListVectorBuckets、プレフィックスの制限、およびレスポンス制限の詳細については、「Amazon S3 API Reference」の「ListVectorBuckets」を参照してください。

プレフィックス検索機能

プレフィックス検索を使用すると、特定のプレフィックスで始まるバケットを一覧表示できるため、関連するベクトルバケットの整理と検索が容易になります。これは、関連するバケットをグループ化する命名規則を使用している場合に特に便利です。

  • 環境ベース: production-vectors-、staging-vectors-、dev-vectors-

  • ユースケースベース: ml-model-vectors-、document-search-、image-similarity-

  • チームベース: data-science-vectors-、ml-platform-vectors-

ベクトルバケットを一覧表示するには

  1. AWS Management Console にサインインし、Amazon S3 コンソール https://console.aws.amazon.com/s3/ を開きます。

  2. ナビゲーションペインで、[ベクトル バケット] を選択します。

    コンソールには、以下の情報を含むすべてのベクトルバケットのリストが表示されます。

    • 名前 – ベクトルバケットの一意の名前

    • 作成日 – バケットが作成された日

    • Amazon リソースネーム (ARN) – プログラムによるアクセスのための完全な ARN

リストをフィルタリングするには

  • バケット名の先頭に基づいてバケットを検索するには、バケットリスト上部の検索ボックスにベクトルバケット名またはプレフィックスを入力します。

  • プレフィックスを使用して関連するバケットのグループを検索します (例えば、すべての本番稼働用バケットを検索するには、「prod-」と入力します)

    入力に応じてリストがリアルタイムで更新されます。
